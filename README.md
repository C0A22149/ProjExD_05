# ゲーム のタイトル
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
簡易版マリオとRPG要素を足したゲーム
・敵を倒すと「コイン」と「スコア」が出る。種類によって固定
・ステージの作成（障害物の地形など）
・「コイン」を消費して、「レベル」や「能力」を獲得できる。
・最終スコアを高いことを目指す。（残り秒数＋「スコア」

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス
* 画面推移機能に関する機能
* フィールドに関するクラス
### 担当追加機能
* 縦長,横長のFieldを約2秒間隔でランダムに表示
